# Простор — лендинг кухонь на заказ (Next.js 16, Tailwind v4)

Премиальный конверсионный лендинг. Стек: Next.js App Router, TypeScript,
Tailwind v4 (CSS-first, токены в `globals.css`), Framer Motion, Phosphor Icons
(`@phosphor-icons/react/dist/ssr`).

## Структура
- `src/lib/content.ts` — весь контент (преимущества, сценарии, акции, тарифы,
  стили, проекты, материалы, этапы, отзывы, FAQ).
- `src/lib/site.ts` — бренд, контакты, навигация, реквизиты.
- `src/lib/phone.ts` — маска/валидация +375.
- `src/lib/kitchen.ts` — модель конструктора (сейчас не на странице).
- `src/components/sections/*` — секции лендинга.
- `src/components/sections/quiz-simple.tsx` — активный квиз-калькулятор (`#quiz`).
- `src/app/api/lead/route.ts` — приём заявок (Telegram через env).

## Tone of voice
Тёплый · ясный · уверенный без пафоса · заботливый · с лёгкой живой ноткой.
Обращение на «вы», активный залог, конкретика и цифры, эмпатия к ситуации.
Не используем: «команда профессионалов», «осуществляем», «под ключ»,
«в кратчайшие сроки», превосходные степени, канцелярит.

## Рабочее правило: проверка
В этой среде превью-скриншоты и eval НИЖЕ первого экрана ненадёжны
(инструмент капчит scroll 0, анимации `whileInView` не триггерятся).
**Не трать ходы на визуальную проверку через preview-скриншоты.**
Достаточно прогнать `npx tsc --noEmit`, `npm run lint`, `npm run build` —
визуальную проверку в браузере делает пользователь сам.
